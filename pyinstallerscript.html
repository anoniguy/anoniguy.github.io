PyInstaller Script

a python script and how to use it

script:

        import PyInstaller.__main__
        import os
        
        app_name = "MyApp"
        
        src_folder = "src"
        app_file = os.path.join(src_folder, "main.py")
        assets_folder = os.path.join(src_folder, "assets")
        
        PyInstaller.__main__.run([
            f'--name={app_name}',
            '--onefile',
            '--windowed',
            '--add-data', f'{assets_folder}:assets',
            app_file,
        ])
