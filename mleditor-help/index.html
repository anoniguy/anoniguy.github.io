MΛ Editor Help

What this is

MΛ Editor is a small Python editor made to be simple and fast. It doesn't try to compete with big IDEs. The goal is just to give you a clean place to write Python code without needing a powerful computer.

Opening and saving files

  • New File (Ctrl + N) creates a blank file
  • Open File (Ctrl + O) lets you load an existing file
  • Open Folder shows a folder in the file tree
  • Save (Ctrl + S) saves your work
  • Save As (Ctrl + Shift + S) saves to a new file

If you open a folder, you can just double-click files from the left side to open them.

The editor

The editor is focused on Python and includes a few helpful things by default:

  • Syntax highlighting for Python
  • Line numbers
  • Auto-closing brackets like (), [] and {}

It behaves like a normal code editor.

Tabs

Every file opens in its own tab. You can switch between them or close them when you're done. If no files are open, you'll see the start screen instead.

Folders

When you open a folder, it appears on the left. From there you can browse files and open them quickly. You can also create new folders inside the current directory using the menu.

Help and documentation

There's a shortcut in the Help menu that opens the official Python documentation in your browser. It's useful if you forget how something works or need examples.
